New Features

  • Select custom audio files from your device
  • Support for MP3, WAV, and OGG formats
  • File size validation (max 10MB)
  • Preview custom sounds before setting alarms
  • Visual indicator for alarms with custom sounds
  • Automatic memory cleanup for deleted alarms
  • Fallback to default beep if custom sound fails

How to Use

  1. Go to the main alarm dashboard
  2. Click "Select Sound File" to choose an audio file
  3. Click "Test Sound" to preview your selection
  4. Set your alarm time and label
  5. Click "Add Alarm" to create the alarm with custom sound
  6. When the alarm triggers, it will play your custom sound

Technical Details

Supported Formats: MP3, WAV, OGG, MP4 (audio)

Maximum File Size: 10MB

Storage: Files are stored as object URLs in browser memory

Persistence: Custom sounds are saved with alarm data

Fallback: Automatic fallback to default beep sounds

Memory Management: Automatic cleanup when alarms are deleted

Important Notes

  • Custom sounds are stored locally in your browser
  • Sounds will be lost if you clear browser data
  • Some browsers may require user interaction to play audio
  • Mobile devices may have different audio behavior
